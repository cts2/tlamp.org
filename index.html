F.A.Q.

Find some answers to the most common questions about TLAMP!

Why are you building TLAMP?

TLAMP is an effort to package standard terminologies into a standards-based terminology service to lower the barrier of entry for individuals looking to get started using terminology in their organizations.

Why is TLAMP distributed as a Virtual Machine image?

TLAMP is a combination of two CTS2 implementations, several supporting tools and utilities, and pre-loaded and processed content. Installation and configuration of these individual components is non-trivial. In order to streamline the process, a ready-to-go instance has been packaged and set up, ready to use.

What is TLAMP's relationship with CTS2?

TLAMP is a CTS2 compliant implementation and all terminology content is represented in CTS2 format.

Specifically, TLAMP is based on a py4cts2, a Python implementation of the CTS2 standard, and a plugin to the CTS2 Framework based on the eXist XML database.

What license does TLAMP have?
